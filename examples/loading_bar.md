# Examples Loading Bar

### Example 1

```txt
>>> p = ProgressBar(5, enabled_average_time=False)
>>> p
[==========........................................]  20.0%

>>> p
[====================..............................]  40.0%

>>> p.display()
[====================..............................]  40.0%

>>> p.format(full="#", empty="_")
>>> p
[##############################____________________]  60.0%
```
